# Installation
 
## Wie installiere ich die App?
Kitamia kann nicht aus dem App-Store bezogen werden. Mehr dazu hier: [Wo finde ich die App?](#wo-finde-ich-die-app)

Die Installation der Kitamia App ist ganz einfach:

**Für Android mit Chrome-Browser:**
1. Öffne die Webseite https://mein.kitamia.de
2. Oben rechts auf die drei Punkte klicken
3. "App installieren" klicken.

**Für Android mit Firefox-Browser:**
1. Öffne die Webseite https://mein.kitamia.de
2. Unten rechts auf die drei Punkte klicken
3. "Installieren" klicken.

**Für iOS mit Safari-Browser:**
1. Öffne die Webseite https://mein.kitamia.de
2. Unten in der Mitte auf den nach oben zeigenden Pfeil klicken
3. "Zum Home-Bildschirm" klicken.
 
## Wo finde ich die App?
Die Kitamia App ist eine [Progressive Web App (PWA)](https://de.wikipedia.org/wiki/Progressive_Web_App). 
Anders als native Apps die aus dem App-Store bezogen werden müssen, setzen wir auf die neue PWA Technologie.

Warum wir das machen?
- Schnellere Updates
- Datenschutz: Wir übermitteln keine Daten von euch an Google oder Apple

Wie die App auf deinem Endgerät installiert werden kann, das beschrieben wir hier: [Wie installiere ich die App?](#wie-installiere-ich-die-app)
